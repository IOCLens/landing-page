### IOCLens - Official Landing Page

This repository hosts the source code for the [IOCLens Official Website](https://ioclens.github.io/landing-page/). 

IOCLens is a privacy-first Chrome extension designed for SOC Analysts to automate IOC enrichment without tab fatigue.

* **[Main Extension Repository](https://github.com/IOCLens/ioclens-for-chrome)** - Source code of the Chrome Extension (Manifest V3).
* **[Chrome Web Store](https://chromewebstore.google.com/detail/ioclens-threat-intel-enri/ileoihlcgdihnnahkdnhebahmljkknnj)** - Install the extension.

###  About this Repository

This repository contains the static assets (HTML/CSS/JS) for the landing page. 
* **Framework:** Pure HTML/CSS (No bloat).
* **Deployment:** Hosted via GitHub Pages.
